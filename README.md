OSBCP CSS Parser
========================================

A simple CSS Parser in Java.


Code example
========================================

	a {
		width: 100px;
		color: red;
	}
	
	b {
		width: 100px;
		color: red;
	}
	
To this:
	
	a, b {
		width: 100px;
		color: red;
	}
